Kids Code Club Full-Stack Dev

Internship

Kids Code Club is a non-profit, government funded coding tutorial website aimed to help children from poor financial backgrounds gain an interest in coding through coding tutorials.

For my first project, I was tasked with adding filter functionality to the tutorials, creating a way for users to select different categories of tutorials to fit their needs.

By altering database tables, schema, models, and controllers within the Laravel Framework, I was able to introduce more functionality into the current tagging system, increasing the unique tag options available for admins on the site. I then incorporated those back-end changes to the site using HTML, PHP, and CSS.

Other tools used in the development of this project were Docker Desktop, Git Bash, and GitHub Desktop, which ensured smooth setup of a development environment, as well as communication with other developers of the site.
