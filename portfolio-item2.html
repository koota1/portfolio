Developing thisPortfolio Website

Personal Project

Creating this portfolio website began from scratch with a blank index.html page. A mostly front=end project, I developed with HTML, CSS, and a small amount of JavaScript. The largest focus of this project was on the CSS.

Incorporating the BEM code naming allows for easy reading and understanding of the CSS document. Using custom properties allows me to make site wide changes such as changing the color scheme, font type, and font weight of every aspect of the website all at once, without having to go into the code and change the styles individually. A heavy focus of the website was to allow for versatility with different screen sizes. Using the @media function, I scaled elements on the page using CSS based on what size the webpage was on the user's end. Using CDNJS and Google APIs allowed for a professional finish in both the font and font consistency of the page, as well as any logos used.
